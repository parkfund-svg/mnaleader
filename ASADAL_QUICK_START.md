# 아사달 호스팅 빠른 배포 가이드

아사달 호스팅에 배포하는 가장 빠른 방법입니다.

## ⚡ 5분 배포 가이드

### 1단계: 아사달 정보 확인 (1분)

아사달 관리자 페이지에서 확인:

1. **FTP 접속 정보**
   - FTP 호스트: `________________`
   - FTP 사용자명: `________________`
   - FTP 비밀번호: `________________`

2. **데이터베이스 정보**
   - 데이터베이스 이름: `________________`
   - 데이터베이스 사용자명: `________________`
   - 데이터베이스 비밀번호: `________________`
   - 데이터베이스 호스트: `localhost` (또는 `________________`)

3. **도메인 정보**
   - 메인 도메인: `https://________________`

### 2단계: config.php 생성 (2분)

1. **파일 복사**
   - `ASADAL_CONFIG_TEMPLATE.php` 파일을 열기
   - 또는 `config.php.example` 파일 복사

2. **정보 입력**
   - 위에서 확인한 정보를 입력:
     ```php
     define('DB_HOST', 'localhost'); // 또는 제공된 호스트
     define('DB_NAME', '실제_데이터베이스명');
     define('DB_USER', '실제_사용자명');
     define('DB_PASS', '실제_비밀번호');
     define('SITE_URL', 'https://실제_도메인.com');
     ```

3. **파일 저장**
   - 이름을 `config.php`로 저장

### 3단계: FTP 업로드 (2분)

1. **FileZilla 실행**
   - FTP 접속 정보 입력
   - 연결

2. **업로드 위치 확인**
   - `public_html` 또는 `www` 폴더로 이동

3. **파일 업로드**
   - 프로젝트 폴더의 모든 파일 선택
   - 드래그 앤 드롭으로 업로드
   - 다음은 제외:
     - `config.php.example`
     - `.git/` 폴더
     - `docker/` 폴더
     - `*.md` 파일

4. **폴더 생성**
   - `uploads/` 폴더 생성 (없는 경우)
   - 권한 755 설정

### 4단계: 데이터베이스 설정 (2분)

1. **phpMyAdmin 접속**
   - 아사달 관리자 페이지 → phpMyAdmin

2. **데이터베이스 선택**
   - 왼쪽에서 데이터베이스 선택

3. **Import 실행**
   - "Import" 탭 클릭
   - `database.sql` 파일 선택
   - "실행" 클릭

### 5단계: 접속 확인 (1분)

1. **메인 페이지**
   - `https://your-domain.com` 접속
   - 정상 표시 확인

2. **관리자 페이지**
   - `https://your-domain.com/admin` 접속
   - 로그인: `admin` / `admin123!`
   - **비밀번호 즉시 변경!**

## 🔧 자주 묻는 질문

**Q: FTP 접속이 안 돼요**
- 포트 번호 확인 (21 또는 22)
- 방화벽 설정 확인
- 아사달 지원팀 문의

**Q: 데이터베이스 연결 오류**
- `config.php`의 정보 확인
- 데이터베이스 호스트 확인
- phpMyAdmin에서 연결 테스트

**Q: 500 에러가 나요**
- `.htaccess` 파일 확인
- PHP 버전 확인 (8.0 이상)
- 에러 로그 확인
